个人环境实验

· 2020-07-22 ·
  • 安装Anaconda最新版
  • 安装pycharm最新版
  • 安装cuda10.2 （因为pytorch官网最新版只支持到10.2）
  • 安装cudnn7.6.5（直接把压缩包中相关文件夹拖到C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.2中对应的目录即可）
  • 根据pytorch官网给定的选项进行安装pytorch
pip install torch===1.5.0 torchvision===0.6.0 -f https://download.pytorch.org/whl/torch_stable.html

以上安装均为下载完成后双击然后下一步就可以。

注意要为cuda的bin目录添加一下环境变量。

遇到问题：cuda死活安装不上（win10 64bit）

尝试方案零（失败）：

直接安装，安装到vs插件的时候会报错一下，继续安装到后面驱动程序会失败，最后全部都失败了。

尝试方案一（失败）：

安全模式使用ddu卸载驱动，然后重启到正常模式安装cuda（期间系统会自动给你安装显卡驱动）。

尝试方案二（失败）：

安全模式使用ddu卸载驱动，然后正常模式中禁止windows update更新驱动，并且不安装vs的插件（还是失败了）

尝试方案三（成功）：

先把驱动卸载了，安装最新的驱动（ 前面组策略设置的禁止更新windows update更新驱动的选项要先关闭），最新驱动安装成功后再安装cuda10.2就直接安装上了（还是要去勾选vs插件）

pycharm中测试pytorch GPU版本是否安装成功：

import torch
from torch import autograd

print(torch.__version__)
print('gpu:', torch.cuda.is_available())
